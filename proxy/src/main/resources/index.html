EnTur Geocoding API

Welcome to the EnTur Geocoding API. This service provides geocoding and reverse geocoding functionality for Norway.

Autocomplete Endpoint

/v2/autocomplete

Search for places by name with autocomplete functionality.

Search for Oslo:
/v2/autocomplete?text=oslo
Search with size limit of 5:
/v2/autocomplete?text=skøyen&size=5
Search with English language:
/v2/autocomplete?text=bergen&lang=en

Reverse Geocoding Endpoint

/v2/reverse

Find places near a specific coordinate.

Lookup Oslo Central Station:
/v2/reverse?point.lat=59.911&point.lon=10.753
Lookup with radius of 200 meters:
/v2/reverse?point.lat=59.911&point.lon=10.753&boundary.circle.radius=200
